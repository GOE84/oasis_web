# OASIS - Innovation for Life

![Oasis Logo](LOGO2.5.png)

A premium single-page e-commerce website and product showcase for **OASIS**, featuring a modern, minimalistic design with smooth animations and interactive elements.

## 🌟 Features

- **Modern Minimalist UI:** Clean aesthetic using Tailwind CSS.
- **Interactive Product Gallery:** Includes a custom zoom-in feature for product images.
- **Single Page Application (SPA):** Smooth "hidden-page" transitions between Home, Products, About, and Contact sections using Vanilla JavaScript.
- **Responsive Design:** Fully optimized for mobile, tablet, and desktop views.
- **Dynamic Content Rendering:** Products and their details are dynamically rendered using Javascript arrays.
